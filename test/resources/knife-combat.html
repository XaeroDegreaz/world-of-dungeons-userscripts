  • When summoning helpers all effect bonuses from items count towards the skill level instead.
  • The attackroll is: 2 x ag + pe + 2 x Knife Combat
  • The defenseroll is: 2 x ag + pe + 2 x Knife Combat
  • The damageroll is: st / 2 + ag / 3 + Knife Combat / 2
  • Each roll will deviate randomly up or down.
  • Some values are shown in color. The colors have the following meanings:
    • Advantage: Values shown in this color can be used to show if something provides a bonus to you or your group (i.e. +1 to an attribute) or a malus (i.e. -1 to an attribute) for your opponent.
    • Disadvantage: Values shown in this color represent a value that is a disadvantage to you or your group, or creates a bonus for your opponent.
    • Bonus/Malus through grafting : This value was caused by grafting ornation material into the item. The standard item has no bonus..
    • Changed by grafting: This value was changed by an ornation material. The standard item has a bonus, but the ornation material has improved (or decreased) the bonus.
